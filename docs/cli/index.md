# CLI reference

The `tet` binary ships with the [tetration](https://github.com/Latka-Industries/tetration) crate.

## Common commands

```bash
# File overview
tet info file.tet

# Query (JSON/TOML expressions — see repo for current syntax)
tet query file.tet '<query>'
```

::: info
Full flag reference will be generated from `--help` output as the CLI stabilizes.
:::

Run `tet --help` for the latest subcommands.
